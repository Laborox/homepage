体験学習教材

Tangie(タンジー)

パズル感覚で、ピースを動かすだけで、プログラミングをする事ができます。

そのプログラムは、即座に他の児童や親が遊ぶことが可能です。

想定児童数 1人～3人(セッションあたり)
対象年齢 6歳～12歳

Fill Craft(フィルクラフト)

タグを児童たちが手書きし、タグを素材として合成する事で発明品をクラフトしていきます。

発明がどう人間の生活に影響を与え、文明が興っていくのかを目のあたりにする事ができます。

推奨児童数 5人～12人
対象年齢 6歳～12歳

予定された出張イベント

ajiromusubi_chirasii.pdf

Tangie 2024年11月2日(文化の日) 網代むすび

Tangieを静岡県熱海市網代952「網代むすび」にて、小学生向けの触って学べるプログラミング講座としてご提供します。プログラミングの事を全く知らない児童を想定しております。

廃校になった網代小学校をリニュアルオープンした学びを起点として、人と人をつなぐことをテーマにした社団法人運営のコミュニティーセンターにて、Tangieを実施提供します。詳しくは => 網代むすびHP

予約制となっており、12名の小学生を体験児童として募集しております。ご予約はこちらまで。


教材の開発方針と心がけている事

創造性を掻き立てる

子どもの視点になって、何に驚くのか、何に感動を覚えるのか、何が創造性を湧き立てるのか？

を常に考えて、セッションを提供します。

遊びと学びの融合

遊び？学び？

学びを最終的な目的としつつ、限りなく遊びに近い形で、提供します。

最近の言葉ではゲーミフィケーションと言います。

触れる入力、魅せる出力

入力は、アナログで触れる…
出力は、デジタルで魅せる…

が、私たちの教材システムの基本思想です。

ゲームのコントローラやタッチパネルを使わず、どんな事ができるのでしょうか？

100% 教育的なゲーム
100% ゲームな教材

私たちは、教材のゲーム的要素においても、教育主体から逸脱しない事を常に心がけています。

全てのゲームが教育的でなく、非教育的なわけでもありません。

ただ、教育的なゲームが教育的なのです。
